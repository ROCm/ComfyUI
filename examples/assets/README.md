# Tutorial input photographs

These three 768x768 PNGs are the real-world inputs that Part 2 of the tutorial
notebook (`amd_comfyui_rocm_tutorial.ipynb`) feeds to **both** generative
workflows: **Hunyuan3D v2.1** (image-to-3D) and **Wan2.2 5B image-to-video**.
Each photograph was sourced from Wikimedia Commons, licence-checked,
centre-cropped to a square, and resized to 768x768 PNG.
